# Social Media Addiction Analysis (Malaysia)

## Project Overview
This project analyses social media addiction levels among Malaysian students and examines its impact on mental health and sleep behaviour using statistical analysis and data visualization.

## Key Insights
- Malaysian students show moderate addiction levels.
- Social media addiction negatively affects mental health.
- Higher addiction levels are associated with reduced sleep hours.

## Tools Used
- Python
- Pandas
- Matplotlib
- Statsmodels
- Google Colab

## Output
A statistical dashboard visualizing addiction trends, mental health correlation, and sleep impact among Malaysian students.
